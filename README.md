## 🚀 Features  
✅ Start, Pause, and Reset stopwatch functionality  
✅ Displays time in **minutes, seconds, and milliseconds**  
✅ Simple and **user-friendly UI**  
✅ Works **smoothly and efficiently**  
✅ No ads, **100% free and open-source**
